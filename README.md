## Présentation projet et problématique :

Pour notre projet de groupe nous avons décidé de travailler sur les données du coronavirus trouvé sur le site data gouv :[Data-gouv-covid19](https://www.data.gouv.fr/fr/datasets/coronavirus-covid19-evolution-par-pays-et-dans-le-monde-maj-quotidienne/#_)

En utilisant le fichier **JSON** à disposition nous allons donc répondre à la problématique suivante. **Comment le nombre de décès et de guéris a-t-il évolué dans le temps au cours de la crise sanitaire ?**
